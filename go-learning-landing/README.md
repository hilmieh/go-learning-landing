## ✨ Fitur Landing Page

- **Hero Section** - Tampilan utama dengan statistik dan preview dashboard
- **Fitur Unggulan** - 6 card fitur utama (Dashboard, Materi, Nilai, Game, Leaderboard, Dark Mode)
- **Mata Pelajaran** - Card produk untuk 5 mapel + Mode Tantangan
- **Cara Bergabung** - 3 langkah simpel mulai belajar
- **FAQ** - Accordion pertanyaan umum
- **CTA Section** - Tombol aksi ke dashboard dan WhatsApp
- **Responsive** - Tampilan optimal di mobile, tablet, dan desktop
- **Animasi** - Scroll reveal, floating cards, smooth transitions

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Font Awesome Icons
- Google Fonts (Plus Jakarta Sans)
- Vanilla JavaScript

## 📦 Cara Deploy ke Vercel

### 1. Fork/Upload ke GitHub
```bash
# Buat repository baru di GitHub
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/go-learning-landing.git
git push -u origin main
```

### 2. Deploy ke Vercel
1. Buka [vercel.com](https://vercel.com) dan login dengan GitHub
2. Klik **"Add New Project"**
3. Pilih repository `go-learning-landing`
4. Klik **"Deploy"**
5. Selesai! Website akan live dalam hitungan detik

## 📝 Cara Edit

Ganti konten sesuai kebutuhan:
